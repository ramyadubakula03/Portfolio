# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my skills, projects, education, and professional profile. The portfolio serves as a central hub where recruiters and employers can learn about my technical expertise, explore my projects, and download my resume.

## 🚀 Live Demo

🔗 **Portfolio Website:**  
https://ramyadubakula03.github.io/Portfolio/

## 📌 Features

- 👋 Professional introduction and profile
- 💼 About Me section
- 🎓 Education details
- 🛠️ Technical Skills
- 📂 Featured Projects
- 📄 Resume download
- 📞 Contact information
- 🔗 GitHub & LinkedIn links
- 📱 Fully Responsive Design
- 🎨 Modern UI with smooth navigation

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Git
- GitHub Pages

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/ramyadubakula03/Portfolio.git
```

Navigate into the project

```bash
cd Portfolio
```

Open the project

```bash
index.html
```

or use **Live Server** in Visual Studio Code.
